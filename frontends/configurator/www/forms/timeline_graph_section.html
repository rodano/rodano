Timeline graph section

"Date" displays a symbol at the specified date.
"Action" displays a vertical line at the specified date.
"Period" displays an horizontal rectangle that begin and end at the specified dates.
"Dot", used with a scale, displays a symbol at the specified date and at the correct vertical position.
"Line" is the same as "Dot" but link all dots with a line.
"Bar" displays a vertical rectangle at the specified date and with the specified value.

For the tooltip, the first line will be used as the title. The following lines are for the content.
Tooltip text will be used as a template literal with the following context: "section" and "entry". For example, you can use ${section.label.en}, ${entry.label}, ${entry.value}, ${entry.date.toDisplay()}, ${entry.date.toFullDisplay()}, ${entry.end_date.toDisplay()}, ${entry.toFullDisplay}, ${entry.link}.
You can also display your meta-data ${entry.metadata.FIELD_MODEL_ID}.
You can also include Javascript code that will be interpreted ${entry.label || section.label.en || ""} ${entry.ongoing ? "-" : entry.end_date.toDisplay()}.
